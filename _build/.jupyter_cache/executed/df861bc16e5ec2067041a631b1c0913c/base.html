\[y(t) =\frac{K\,{\mathrm{e}}^{-\omega_n \,t\,\zeta } \,\sin \left(\omega_n \,t\,\sqrt{1-\zeta^2 }\right)}{\omega_n \,\sqrt{1-\zeta^2 }}\]
